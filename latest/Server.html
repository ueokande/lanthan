Class: Server

Server(address, port, client, listener, logger)

Server is an remote WebExtensions API server. The server is a proxy for between firefox and the API client. The server listen on HTTP port to receive a HTTP request to invoke WebExtensions API. The result of it is returned on the HTTP response. The server communicate with a browser via native messaging.
